# Wusiit — What split is it today?

Small frontend app to track workout split days and record per-day descriptions. Local-first, no backend required.

Quick start

1. Install deps

```bash
npm install
```

2. Run dev server

```bash
npm run dev
```

3. Open http://localhost:5173

Notes
- Configure your split in the "Split" view (comma-separated).
- Use "Mark Done" on Home to record today's workout and add a description.
- Export/Import JSON in Settings for backups.
